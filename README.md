# 🧠 Design Patterns Guide (Spring Boot Examples)

This project is a simple, easy-to-follow guide that explains common software design patterns using Java and Spring Boot examples.

It focuses on helping developers understand:

- What each design pattern does
- When to use it
- How it looks in real-world backend development

The goal is to make design patterns more practical and easier to apply, especially for those working with modern frameworks like Spring Boot.

---

## 🤖 Note

This guide was created with the help of AI to organize concepts, provide examples, and make learning more accessible.

## 🚀 Getting Started

This is a Next.js project bootstrapped with `create-next-app`.

### Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Then open http://localhost:3000 in your browser to see the app.

The page will automatically reload when you make changes.
